git-open
========

git open [~/path/to/repo]

Usage
-----

```
git-open [~/path/to/repo]
git open [~/path/to/repo]
```

Requirements
------------

- git command (in PATH)

Install
-------

For Go user:

```
go get -u github.com/mrluanma/git-open
```

For others:

[Downloads release binary and put it in your PATH](http://gobuild.io/download/github.com/mrluanma/git-open)
